# САПР (CAD)

На этой странице представлены некоторые из моих CAD разработок.

## Быстро-сменный захват для робота

Данная разработка вдохновлена заданием по компетенции "Прототипирование и обслуживание мобильных робототехнических систем" 
Всероссийского чемпионатного движения по профессиональному мастерству «ПРОФЕССИОНАЛЫ».

Для выполнения задания робот должен имет две системы менеджмента объектов (СМО), каждая из которых предназначена для выполнения
различных задач, например захват муляжей фруктов и загрузка и выгрузка семян из дозатора в соты.

На фото с левой стороны база с установленым сервопроводом, а справа насадка на которую передается крутящийся момент.
Взаимное крепление осуществляется благодаря скобы.
