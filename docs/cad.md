## Система быстрой замены манипулятора робота

Данная разработка вдохновлена заданием по компетенции "Прототипирование и обслуживание мобильных робототехнических систем" 
Всероссийского чемпионатного движения по профессиональному мастерству «ПРОФЕССИОНАЛЫ».

Для выполнения задания робот должен имет две системы менеджмента объектов (СМО), каждая из которых предназначена для выполнения
различных задач, например захват муляжей фруктов и загрузка семян из дозатора и их выгрузка в соты.

Модель спроектирована программой AudoCAD (.dwg), преобразована программой UltiMaker Cura в формат для печати на 3D принтере (.stl).
На фото с левой стороны база с установленым сервопроводом, которая крепиться к роботу, а справа насадка на которую передается крутящийся момент.
Крепление насадки к базе осуществляется скобой (здесь не показана), которую можно легко снять и установить насадку с другим механизмом.
