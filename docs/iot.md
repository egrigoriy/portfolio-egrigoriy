## Шагомер с веб логированием

Это проект из курса 6.08 "Взаимосвязаные встроеные системы" Массачусетского технологического института.

Цель состоит в разработке системы, которая должна:

1. считать шаги;
2. при нажатии одной кнопки будет отправлять POST-запрос с текущем количеством шагов пользователя в базу данных на сервер и обнулять текущее значение;
3. использовать вторую кнопку для отображения текущего локального количества шагов или общего количества пройденных шагов с сервера. 

Использованы:  
1. контроллер ESP32
2. 3-х осевой гироскоп и акселерометр GY-521 (MPU 6050) - интерфейс I2C 
3. TFT ЖК-дисплей 1.8 дюйма ST7735, 128х160- интерфейс SPI
