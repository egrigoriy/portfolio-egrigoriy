## Gitlet

Gitlet - это проект из курса CS61B "Структуры данных" Калифорнийского Университета в Беркли.  

Это консольное приложение представляющее систему контроля версий файлов и имитирующие основные функции популярной системы Git.  

### Требуемый функционал:  

#### <u>Для работы с локальным репозитарием:</u>  

- init
- add
- commit
- rm
- log
- find
- status
- checkout
- branch
- rm-branch
- reset
- merge

#### <u>Для работы с удаленным репозитарием:</u>  

- add-remote
- rm-remote
- push
- fetch
- pull  

### Тестирование работы

## CS61BYoW

CS61BYoW - это проект из курса CS61B "Структуры данных" Калифорнийского Университета в Беркли.

Проект создания игры из двух частей:  

1. игровой движок с генерацией псевдослучайного мира в виде лабиринта

2. интерактивный консольный интерфейс - меню создания новой игры/загрузки сохранненной и режим игры с управленим героя из клавишей

## От NAND вентиля до ТЕТРИСА  

[Курс](https://www.nand2tetris.org/.) по книге “The Elements of Computing Systems” за авторством Noam Nisan и Shimon Schocken 
Начиная только с логических элементов NAND, требуется создать аппаратную часть (АЛУ, ЦП), 
а затем программное обеспечение (компилятор, ассемблер, ОС), чтобы в конечном итоге запустить такую ​​игру, как Тетрис.  

![nand2tetris.png](/public/nand2tetris.png)  

### От NAND вентиля до Ассемблера  

В первой части курса стоит задача построит вычислительную машины под названием HACK.

![hack1.png](/public/hack1.png)

Внутреняя реализация компьютера выгладит так:  


![hack2.png](/public/hack2.png)  


Внутреняя реализация центрального процессора выгладит так:  
 

![hack_cpu.png](/public/hack_cpu.png)  

Все элементы компьютера, от вентилей, арифметическо-логическое устройство и память, описываются на языке VHDL.
Заканчивается первая часть написанием ассемблере для используя следующую архитектуру набора команд:  

![hack_isa.png](/public/hack_isa.png)

Ассемблер был написан на Java. Все тесты пройдены успешно.
Ссылка на репозитарий моего решения: [Github](https://github.com/egrigoriy/hackassembler)

### От Операционной системы до Ассемблера

Вторая части курса покрывавет разработку программного обеспечения.
Она включает написание примитивных функции операционной системы, 
компилятор для языка Jack (Java-подобый) в байт-код для виртуальной машины и перевод байт-кода в ассемблер.
 
![nand2tetris2.png](/public/nand2tetris2.png)  

Мой вариант компилятора был написан на Java.
Ссылка на репозитарий: [Github](https://github.com/egrigoriy/jackcompiler)

Мой вариант компилятора байт-кода в ассемблер был написан на Java.
Ссылка на репозитарий: [Github](https://github.com/egrigoriy/vmtranslator)
