# Мобильные приложения

На этой странице представлены некоторые из мобильных приложений написаные мною на языке программирования Kotlin.

## QR сканнер с передачей данных на сервер

Это мобильное приложение позволяет с помощью телефона считывать штих и QR коды и отправлять данные на сервер.

Функционал:
1. подключение к серверу по считаному QR коду
2. сканирование штих или QR код маркированного предмета
3. выбор количества предметов с данной маркировкой
4. отправление кода и количества предметов на сервер
